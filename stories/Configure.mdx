# VG Component Library

**A modern TypeScript-based component library built with Lit 3**

VG is a comprehensive design system featuring theme-aware web components with seamless framework integrations for React, Vue, Angular, and vanilla HTML/JavaScript.

---

## üöÄ Quick Start

### Installation

```bash
# From GitHub repository
npm install https://github.com/brguru90/lit-components.git
```

**Or add to package.json:**

```json
"dependencies": {
  "vg": "github:brguru90/lit-components"
}
```

Then run `npm install`

### Basic Usage

```html
<!DOCTYPE html>
<html>
<head>
  <script type="module" src="./node_modules/vg/dist/index.js"></script>
  <link rel="stylesheet" href="./node_modules/vg/dist/index.css" />
</head>
<body>
  <vg-theme-provider mode="dark">
    <vg-card heading="Welcome to VG">
      <vg-button variant="primary" size="md">Get Started</vg-button>
    </vg-card>
  </vg-theme-provider>
</body>
</html>
```

## üîß Framework Integration

### React Integration
```jsx
import { VgThemeProvider, VgButton, VgCard } from "vg/react";
import "vg/index.css";

function App() {
  return (
    <VgThemeProvider mode="dark">
      <VgCard heading="React Example">
        <VgButton 
          variant="primary"
          onVgClick={(event) => console.log('Clicked!', event.detail)}
        >
          Click Me
        </VgButton>
      </VgCard>
    </VgThemeProvider>
  );
}
```

### Vue Integration
```vue
<template>
  <vg-theme-provider mode="dark">
    <vg-card heading="Vue Example">
      <vg-button variant="primary" @vg-click="handleClick">
        Click Me
      </vg-button>
    </vg-card>
  </vg-theme-provider>
</template>

<script setup>
import "vg/vue";
const handleClick = (event) => console.log('Clicked!', event.detail);
</script>
```

### Angular Integration
```typescript
// app.component.ts
import { Component, CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';

@Component({
  selector: 'app-root',
  template: `
    <vg-theme-provider mode="dark">
      <vg-card heading="Angular Example">
        <vg-button variant="primary" (vg-click)="handleClick($event)">
          Click Me
        </vg-button>
      </vg-card>
    </vg-theme-provider>
  `,
  schemas: [CUSTOM_ELEMENTS_SCHEMA]
})
export class AppComponent {
  handleClick(event: Event) {
    console.log('Clicked!', (event as CustomEvent).detail);
  }
}
```

## üìã Event System

All VG components follow a consistent event naming convention:

- Events are prefixed with `vg-` (e.g., `vg-click`, `vg-change`)
- Framework wrappers provide appropriate event handlers:
  - **React**: `onVgClick`, `onVgChange`
  - **Angular**: `(vg-click)`, `(vg-change)`  
  - **Vue**: `@vg-click`, `@vg-change`
